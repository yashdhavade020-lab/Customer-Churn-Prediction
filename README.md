# Customer Churn Prediction

Machine Learning project to predict customer churn using Random Forest.

## Features
- Data preprocessing
- Exploratory Data Analysis
- SMOTE balancing
- Random Forest model
- Streamlit web app

## Files
- app.py
- customer_churn_data.csv
- new_nb.ipynb
- randomforest_churn_model.pkl

## Run

```bash
pip install streamlit pandas scikit-learn
streamlit run app.py
```
